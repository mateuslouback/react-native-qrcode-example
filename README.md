# QR Code generator App

Example App to generate QR Code to use **React Native**.

![](gif.gif)

## Package installation

Install Yarn:
Open (https://yarnpkg.com/).

Run terminal code:

```
$ yarn install
```

## Run app in IOS

Run terminal code:

```
$ react-native run-ios
```

## Run app in Android

Run terminal code:

```
$ react-native run-android
```
